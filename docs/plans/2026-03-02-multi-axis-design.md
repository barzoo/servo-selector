# 多轴伺服选型功能设计文档

**日期**: 2026-03-02
**功能**: 支持用户配置多个伺服轴，管理轴列表，导出整合PDF报告
**设计**: 响应式侧边栏方案

---

## 1. 概述

### 1.1 目标
- 允许用户在一个项目中配置多个伺服轴
- 每个轴独立进行选型计算
- 支持轴的增删改查
- 导出整合的多轴PDF报告

### 1.2 核心原则
- **渐进式**: 单轴用户不受影响
- **响应式**: 桌面端侧边栏，移动端抽屉
- **状态可见**: 每个轴的状态（配置中/已完成）一目了然

---

## 2. 数据模型

### 2.1 核心类型

```typescript
// 轴的状态
type AxisStatus = 'CONFIGURING' | 'COMPLETED' | 'ABANDONED';

// 单个轴的数据
interface AxisConfig {
  id: string;                    // 唯一标识
  name: string;                  // 用户自定义名称
  status: AxisStatus;
  createdAt: string;             // ISO 时间戳
  completedAt?: string;          // 完成时间
  input: Partial<SizingInput>;   // 选型输入数据
  result?: SizingResult;         // 计算结果
  selectedMotorIndex?: number;   // 用户选择的电机索引
}

// 项目（包含多个轴）
interface Project {
  id: string;
  name: string;
  customer: string;
  salesPerson: string;
  notes?: string;
  createdAt: string;
  axes: AxisConfig[];
}

// 扩展 WizardState
interface WizardState {
  currentStep: WizardStep;
  isComplete: boolean;
  project: Project;
  currentAxisId: string;
  input: Partial<SizingInput>;
  result?: SizingResult;
}
```

### 2.2 存储键名
- `servo-selector-project`: 新项目数据
- `servo-selector-wizard`: 旧数据（迁移后弃用）

## 5. 关键交互流程

### 5.1 添加新轴
1. 用户在结果页点击"💾 保存到篮子"
2. 当前轴状态变为 `COMPLETED`
3. 自动弹出选项：
   - "🔄 基于此轴创建新轴"
   - "➕ 添加空白新轴"
   - "📋 查看项目篮子"

### 5.2 切换轴
1. 用户点击侧边栏中的其他轴
2. 如果当前轴有未保存更改，弹出确认对话框
3. 加载选中轴的数据到表单
4. 根据轴状态跳转页面

### 5.3 删除轴
1. 轴项悬停/长按显示删除按钮
2. 确认删除后从列表移除
3. 如果删除的是当前轴，切换到第一个可用轴

function migrateLegacyData(): Project | null {
  const legacy = localStorage.getItem('servo-selector-wizard');
  if (!legacy) return null;

  const parsed = JSON.parse(legacy);

  return {
    id: generateId(),
    name: parsed.input?.project?.name || '未命名项目',
    customer: parsed.input?.project?.customer || '',
    salesPerson: parsed.input?.project?.salesPerson || '',
    createdAt: new Date().toISOString(),
    axes: [{
      id: generateId(),
      name: '轴-1',
      status: parsed.isComplete ? 'COMPLETED' : 'CONFIGURING',
      createdAt: new Date().toISOString(),
      completedAt: parsed.isComplete ? new Date().toISOString() : undefined,
      input: parsed.input,
      result: parsed.result,
    }],
  };
}

## 9. 错误处理

| 场景 | 处理 |
|------|------|
| 删除最后一个轴 | 不允许，至少保留一个轴 |
| 切换轴时未保存 | 确认对话框 |
| 项目名称重复 | 自动生成"轴-1"、"轴-2" |
| 无已完成轴时导出 | 禁用按钮，提示用户 |
| localStorage 满 | 压缩数据或提示清理 |
