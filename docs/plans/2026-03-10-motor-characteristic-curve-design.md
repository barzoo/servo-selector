# 电机特性曲线与工况需求对比图设计文档

> **设计日期**: 2026-03-10
> **功能**: 在结果步骤展示电机特性曲线与工况需求的对比，帮助用户直观理解选型匹配度

---

## 1. 需求背景

原速度-时间曲线存在以下问题：
- 只展示理想运动学曲线，与电机选型决策无关
- S曲线形状不正确
- 无法帮助用户理解选型结果

**核心洞察**：用户需要理解的是**电机能否满足工况要求**，而非运动学曲线形状。

---

## 2. 设计目标

- **直观展示**电机能力边界（特性包络线）
- **清晰对比**实际工况需求与电机极限
- **一目了然**各阶段安全余量

---

## 3. 组件位置

将图表放置在**结果步骤 (ResultStep)** 的推荐电机详情区域：

```

├─────────────────────────────────────────────┤
```

## 4. 技术方案

### 4.1 实现方式：SVG 动态绘制

选择 SVG 方案的原因：
- 无需额外依赖
- 精确控制视觉呈现
- 与现有代码风格一致
- 文件体积小

### 4.2 图表组成

**1. 电机特性包络线**（灰色填充区域）
- 从 (0, 峰值扭矩) 到 (额定转速, 峰值扭矩)
- 从 (额定转速, 峰值扭矩) 到 (最大转速, 额定转矩)
- 从 (最大转速, 额定转矩) 到 (最大转速, 0)
- 闭合到原点

**2. 额定转矩参考线**（虚线）
- 水平线，标记电机额定转矩

**3. 三个工况点**（彩色圆点）
- 绿色：加速点 (加速转速, 加速转矩)
- 蓝色：恒速点 (最大转速, 恒速转矩)
- 橙色：减速点 (加速转速, 减速转矩)

**4. 坐标轴**
- X轴：转速 (rpm)
- Y轴：转矩 (N·m)

### 4.3 视觉样式

| 元素 | 样式 |
|------|------|
| 包络线填充 | `var(--foreground-muted)` 10% 透明度 |
| 包络线边框 | `var(--foreground-muted)` 30% 透明度 |
| 额定转矩线 | `var(--primary-400)` 虚线 |
| 加速点 | `var(--green-400)` 8px 圆点 |
| 恒速点 | `var(--blue-400)` 8px 圆点 |
| 减速点 | `var(--amber-400)` 8px 圆点 |
| 坐标轴 | `var(--foreground-muted)` |

## 5. 组件接口

```typescript
interface MotorCharacteristicChartProps {
  // 电机参数
  motor: {
    ratedTorque: number;      // N·m
    peakTorque: number;       // N·m
    ratedSpeed: number;       // rpm
    maxSpeed: number;         // rpm
  };

  // 工况需求
  operatingPoints: {
    accel: { speed: number; torque: number };   // 加速点
    constant: { speed: number; torque: number }; // 恒速点
    decel: { speed: number; torque: number };   // 减速点
  };

  className?: string;
}
```

---

## 6. 数据来源

从 `SizingResult` 中提取：
- `mechanical.torques.accel` - 加速转矩
- `mechanical.torques.constant` - 恒速转矩
- `mechanical.torques.decel` - 减速转矩
- `mechanical.speeds.max` - 最大转速
- `motorRecommendations[0].motor` - 推荐电机参数

---

## 7. 边界情况处理

- **无推荐电机**：不显示图表
- **工况点超出包络线**：红色警告样式
- **电机参数缺失**：显示占位符
