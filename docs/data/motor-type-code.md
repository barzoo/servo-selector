# MC20系列永磁同步伺服电机订货号编码规则

## 型号格式
```
MC20-{法兰尺寸}-3{转子惯量}{额定转速}-N{功率编码}-{电机制造年份}{编码器类型}{防护等级}{温度传感器}-{电机轴}{电气连接}{编码器规格}
```

**完整示例：** `MC20-060-3M20-N102-0APL-NNNN`

## 编码规则详细说明

### 位段1: 产品系列 (固定)
```
MC20
```

### 位段2: 法兰尺寸
| 代码 | 尺寸 (mm) |
|------|-----------|
| 060  | 60.0 × 60.0 |
| 080  | 80.0 × 80.0 |
| 100  | 100.0 × 100.0 |
| 130  | 130.4 × 130.4 |
| 180  | 180.0 × 180.0 |

### 位段3: 电源电压 (固定)
```
3 = 三相交流380V
```

### 位段4: 转子惯量等级
| 代码 | 惯量等级 |
|------|----------|
| L    | 低惯量   |
| M    | 中惯量   |
| H    | 高惯量   |

### 位段5: 额定转速
| 代码 | 转速 (rpm) |
|------|------------|
| 15   | 1500       |
| 20   | 2000       |
| 30   | 3000       |

### 位段6: 冷却方式 (固定)
```
N = 自然冷却
```

### 位段7: 额定功率编码
| 功率编码 | 额定功率 |
|----------|----------|
| 201      | 200W     |
| 401      | 400W     |
| 751      | 750W     |
| 851      | 0.85kW   |
| 102      | 1kW      |
| 132      | 1.3kW    |
| 152      | 1.5kW    |
| 182      | 1.8kW    |
| 202      | 2.0kW    |
| 252      | 2.5kW    |
| 292      | 2.9kW    |
| 302      | 3.0kW    |
| 352      | 3.5kW    |
| 402      | 4.0kW    |
| 442      | 4.4kW    |
| 502      | 5kW      |
| 552      | 5.5kW    |
| 752      | 7.5kW    |


### 位段8: 电机制造年份
| 代码 | 说明   |
|------|--------|
| 0    | 无刹车 |
| 1    | 有刹车 |

### 位段9: 编码器类型
| 代码 | 编码器规格 |
|------|------------|
| A    | 23位多圈绝对值编码器（电池盒式多圈，2.5Mbps多摩川协议） |
| B    | 23位多圈绝对值编码器（机械式多圈，5Mbps多摩川协议） |

### 位段10: 防护等级
| 代码 | 防护等级 |
|------|----------|
| P    | IP65（带插封） |
| L    | 其他等级 |

### 位段11: 温度传感器
| 代码 | 温度传感器 |
|------|------------|
| N    | 无         |

### 位段12: 电机轴
| 代码 | 轴型 |
|------|------|
| L    | 光轴 |
| K    | 键槽 |

### 位段13: 电气连接
| 代码 | 连接方式 |
|------|----------|
| C    | 简易插头 |
| P    | 航空插头 |

### 位段14: 编码器规格 (4位)
```
NNNN = 根据具体编码器规格填写，通常为4位数字或字母数字组合
```

### 编码生成函数逻辑
1. 验证输入参数的有效性
2. 根据功率需求查找对应的功率编码
3. 按位段顺序拼接各部分代码
4. 返回完整的订货号

### 注意事项
- 电源电压和冷却方式为固定值
- 温度传感器当前只支持"无"选项
- 编码器规格部分需要根据具体产品手册确定
- 建议在程序中加入参数兼容性验证逻辑
