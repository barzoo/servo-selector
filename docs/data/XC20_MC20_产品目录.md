系统包括：
- XC20伺服驱动器

## XC20伺服驱动型号命名规则

### 完整订货号结构

示例型号：XC20-W0005CRN-01BECTONNNN-SVSRSN3TEONNN

分段结构：XC20-W[电流][IP][制动][预留]-[硬前缀][面板][总线][安全][硬预留]-[固件版本][固预1][总线][PLC][固预2]

### 第1段：功率部分

格式：XC20-W[4位电流][1位IP][1位制动][1位预留]

示例：XC20-W0005CRN

| 位置 | 代码 | 含义 |
|---|---|---|
| 1-4 | XC20 | 产品类型：新一代经济型伺服驱动器 |
| 5 | W | 功率段标识（固定字符W） |
| 6-9 | 0005 | 最大5A，额定1.5A |
| | 0007 | 最大7A，额定2A |
| | 0012 | 最大12A，额定4A |
| | 0023 | 最大23A，额定6.3A |
| | 0033 | 最大33A，额定10A |
| | 0050 | 最大50A，额定16A |
| 10 | C | IP20，200-440V三相交流（宽幅输入+10%/-15%） |
| 11 | R | 内置制动电阻 |
| | N | 无内置制动电阻 |
| 12 | N | 功率段预留位（默认N） |

### 第2段：控制部分-硬件选项

格式：[2位前缀][1位面板][2位总线][2位安全][4位预留]

示例：01BECTONNNN

| 位置 | 代码 | 含义 |
|---|---|---|
| 1-2 | 01 | 硬件版本前缀（固定01） |
| 3 | B | 面板：显示屏（5位7段，2LED，3按钮） |
| | N | 面板：无显示屏 |
| 4-5 | EC | 总线接口：EtherCAT |
| | PN | 总线接口：ProfiNet IO |
| | EI | 总线接口：EthernetIP |
| | NN | 总线接口：无 |
| 6-7 | T0 | 安全功能：安全扭矩关断(STO) |
| | NN | 安全功能：无安全选项 |
| 8-11 | NNNN | 硬件预留位（默认NNNN） |

### 第3段：控制部分-固件选项

格式：[5位版本][1位预留][1位总线][3位PLC][3位预留]

示例：SVSRSN3TEONNN

| 位置 | 代码 | 含义 |
|---|---|---|
| 1-5 | SVSRS | 固件版本：目前最新版 |
| 6 | N | 固件预留位1（默认N） |
| 7 | 2 | 总线通信：EtherCAT (SoE) |
| | 3 | 总线通信：EtherCAT (CoE) |
| | 4 | 总线通信：PROFINET IO |
| | 5 | 总线通信：Ethernet IP |
| 8-10 | TEO | PLC功能：带PLC功能 |
| | NNN | PLC功能：无PLC功能 |
| 11-13 | NNN | 固件预留位（默认NNNN） |

### 选型匹配说明

硬件总线接口(EC/PN/EI)与固件总线通信(2/3/4/5)需匹配：
- EtherCAT硬件(EC)对应固件2(SoE)或3(CoE)
- ProfiNet硬件(PN)对应固件4
- EthernetIP硬件(EI)对应固件5
