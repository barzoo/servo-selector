# XC20伺服驱动系统 & MC20伺服电机 产品目录

## 产品概述

XC20伺服驱动系统是ctrlX AUTOMATION家族的重磅新成员，满足本土市场在性能与成本上的双重期待，是您的极"智"之选！

系统包括：
- XC20伺服驱动器
- MC20伺服电机
- 电缆及附件

**功率范围**: 0.2kW 至 7.5kW  
**安全功能**: 配备STO安全功能  
**通信选项**: EtherCAT、Profinet、Ethernet IP等多种通信选项  
**设计目标**: 专为满足各种标准应用而设计

- 主轴定位
- 外部扭矩前馈
- 参数切换(星角切换、齿轮箱切换等)

### 补偿/修正
- 反向间隙补偿
- 摩擦力修正
- 重量负载补偿
- 自动重量负载补偿
- 温度误差补偿
- 扭矩常数补偿
- 齿槽补偿
- 自适应速度环路增控制益
- 混合速度控制(速度混合)
- 混合位置控制

- 保证在恶劣的现场环境中稳定运行
- F级绝缘绕组
- IP65防护等级
- 标配轴封

- 标配电子铭牌功能

## XC20伺服驱动型号命名规则

### 型号结构
```
XC20-WXXXXCRN-XXBXXXXNNNN-SVSRSNXXXXXX
```

### 各字段含义

| 位置 | 代码 | 含义 |
|------|------|------|
| **产品类型** | XC20 | 新一代经济型伺服驱动器 |
| **最大电流** | W0005-W0050 | 最大5A-50A，额定1.5A-16A |
| **IP等级和输入电压范围** | CRN | C=IP20, 200-440V三相交流宽幅输入(+10%/-15%) |
| **控制部分** | 01BECT0NNNN | 01=面板类型，B=显示屏，ECT0=EtherCAT (CoE)，NNNN=不带安全选项 |
| **硬件选项** | SVSRS | 制动电阻(R=内置) |
| **固件选项** | N3TE0NNN | 固件版本 |

### 详细配置选项

**最大电流**:
- 0005 = 最大5A，额定1.5A
- 0007 = 最大7A，额定2A
- 0012 = 最大12A，额定4A
- 0023 = 最大23A，额定6.3A
- 0033 = 最大33A，额定10A
- 0050 = 最大50A，额定16A

**IP等级和输入电压范围**:
- C = IP20, 200-440V三相交流，宽幅输入(+10%/-15%)

**面板**:
- B = 显示屏，5位7段，2个LED灯，3个按钮

**制动电阻**:
- R = 内置制动电阻

**总线接口类型**:
- PN = ProfiNet IO
- EC = EtherCAT
- EI = EthernetIP（准备中）

**总线通信**:
- 2 = EtherCAT (SoE)
- 3 = EtherCAT (CoE)
- 4 = PROFINET IO
- 5 = Ethernet IP

**安全功能选项**:
- T0 = 安全扭矩关断(STO)
- NN = 不带安全选项

**固件版本**:
- SVSRS = 目前最新版

**驱动器可集成PLC功能**:
- TE0 = 带PLC（准备中）
- NNN = 无
