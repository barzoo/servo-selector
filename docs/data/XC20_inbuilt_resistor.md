# XC20 内置制动电阻选型指南

## 订货号标识规则

在 XC20 订货号中，**第 11 位字符**标识制动电阻配置：

| 字符 | 含义 |
|:---|:---|
| **R** | 内置制动电阻（本文档适用范围） |
| **N** | 无内置制动电阻（需外接制动电阻） |

订货号示例：`XC20-W0005CRN` → 内置制动电阻（160Ω，40W）

### 关键参数定义

- **持续功率 (40W/80W)**：制动电阻可长期连续耗散的功率。超过此值会导致电阻过热，驱动器报过温故障。
- **峰值功率**：短时（通常秒级）可承受的最大制动功率，与减速瞬间的能量冲击有关。
- **开启阈值**：母线电压达到 (U<sub>LN</sub> × √2 × 1.05 + 15) V 至 800V 时开启制动（电压相关模式为 770-790V）。

## 内置制动电阻适用性判断

### 计算步骤

1. **计算单次制动能量**  
   E = ½ × J × (ω<sub>1</sub>² - ω<sub>2</sub>²)  
   - J：总转动惯量（电机惯量 + 负载惯量，单位 kg·m²）
   - ω：角速度（rad/s），ω = 2π × n / 60，n 为转速（rpm）

2. **计算平均制动功率**  
   P<sub>avg</sub> = (E × N) / 60  
   - N：每分钟制动次数（次/分钟）
   - 结果单位：W（焦耳/秒）

3. **对比判断**  
   - 若 P<sub>avg</sub> ≤ **40W**（0005/0007/0012 型号）或 ≤ **80W**（0023/0033/0050 型号），内置电阻可用。
   - 若 P<sub>avg</sub> > 持续功率，必须选用外接制动电阻或降低制动频率。



**注意**：若计算显示平均功率超过持续功率（40W/80W），即使单次制动峰值未超限，也必须改用外接制动电阻或增加制动电阻散热措施。
