# 机械负载类型与选型计算详细规格

**版本**: 1.0
**日期**: 2026-02-27
**适用范围**: XC20 + MC20 伺服系统选型工具

---

## 1. 概述

本文档定义伺服选型工具支持的机械负载类型、物理模型、计算公式及参数验证规则。所有算法均基于经典力学和伺服选型工程实践。

### 1.1 负载类型总览

| 类型 | 代码 | 典型应用 | 复杂度 |
|------|------|----------|--------|
| 滚珠丝杠 | BALL_SCREW | CNC机床、3D打印机、精密定位 | 高 |
| 齿轮/减速机 | GEARBOX | 工业机器人、转盘、AGV | 高 |
| 直接驱动 | DIRECT_DRIVE | 直驱电机、DDR转台 | 低 |
| 同步带/皮带 | BELT | 输送线、轻型定位 | 中 |
| 齿条齿轮 | RACK_PINION | 大型龙门、激光切割机 | 中 |
| 摆臂/连杆 | LINKAGE | 包装机械、拾取放置 | 高 |

## 2. 滚珠丝杠传动 (BALL_SCREW)

### 2.1 物理模型

```
    电机 → 联轴器 → 丝杠 ──→ 螺母 → 工作台(负载)
                    ↑
                 导程 Pb (mm/rev)
```

### 2.2 输入参数

| 参数 | 符号 | 单位 | 必填 | 默认值 | 范围 |
|------|------|------|------|--------|------|
| 工作台质量 | m | kg | 是 | - | 0.1 - 10000 |
| 丝杠导程 | Pb | mm | 是 | - | 1 - 100 |
| 丝杠直径 | d | mm | 否 | 0.3×Pb | 5 - 100 |
| 丝杠长度 | L | mm | 否 | - | 50 - 5000 |
| 减速比 | i | - | 否 | 1 | 1 - 50 |
| 机械效率 | η | % | 否 | 90% | 50% - 95% |
| 摩擦系数 | μ | - | 否 | 0.05 | 0.001 - 0.3 |
| 预压扭矩 | Tp | N·m | 否 | 0 | 0 - 10 |
| 安装方向 | - | - | 是 | 水平 | 水平/垂直向上/垂直向下 |

### 2.3 惯量计算

**总惯量 = 负载惯量 + 丝杠惯量**

```typescript
// 负载惯量 (转换为电机轴)
J_load = m × (Pb / (2π × i))² × 10⁻⁶  // kg·m²

// 丝杠惯量 (实心圆柱)
J_screw = (π × ρ_steel × L × d⁴) / 32 × 10⁻¹⁵  // kg·m²
// ρ_steel = 7850 kg/m³

// 总负载惯量
J_total = J_load + J_screw / i²
```

**参考值**: 滚珠丝杠效率 η = 0.90 - 0.95

### 2.4 扭矩计算

#### 2.4.1 加速扭矩

```typescript
// 线性加速度转换为角加速度
α = a × (2π × i) / (Pb × 10⁻³)  // rad/s²

// 加速扭矩
T_accel = (J_motor + J_total) × α / η  // N·m
```

#### 2.4.2 摩擦扭矩

```typescript
// 导轨摩擦力
F_friction = μ × m × g  // N

// 摩擦扭矩
T_friction = F_friction × (Pb × 10⁻³) / (2π × η × i)  // N·m
```

#### 2.4.3 重力扭矩（垂直安装）

```typescript
// 垂直轴重力
F_gravity = m × g  // N

// 重力扭矩（始终与运动方向相反）
T_gravity = F_gravity × (Pb × 10⁻³) / (2π × i)  // N·m

// 向上加速时
T_up = T_accel + T_friction + T_gravity

// 向下加速时
T_down = T_accel + T_friction - T_gravity  // 注意：可能为负（再生状态）
```

#### 2.4.4 预压扭矩（如有）

```typescript
// 预压产生的恒定扭矩
T_preload = Tp / i  // N·m
```

### 2.5 速度换算

```typescript
// 线性速度 → 电机转速
n = v × 60 × i / (Pb × 10⁻³)  // rpm

// 电机转速 → 线性速度
v = n × (Pb × 10⁻³) / (60 × i)  // m/s
```

## 3. 齿轮/减速机传动 (GEARBOX)

### 3.1 物理模型

```
    电机 → 减速机(i) → 输出轴 → 负载转盘/滚筒
```

### 3.2 输入参数

| 参数 | 符号 | 单位 | 必填 | 默认值 | 范围 |
|------|------|------|------|--------|------|
| 负载类型 | - | - | 是 | 转盘 | 转盘/滚筒/其他 |
| 负载质量 | m | kg | 是 | - | 0.1 - 50000 |
| 减速比 | i | - | 是 | - | 1 - 1000 |
| 转盘直径 | D | mm | 条件 | - | 10 - 5000 |
| 滚筒直径 | D | mm | 条件 | - | 10 - 2000 |
| 机械效率 | η | % | 否 | 见表 | 30% - 98% |
| 摩擦扭矩 | Tf | N·m | 否 | 0 | 0 - 1000 |
| 重力臂长 | Lg | mm | 否 | 0 | 0 - 5000 |

### 3.3 减速机效率参考

| 类型 | 减速比范围 | 效率 | 回程间隙 |
|------|-----------|------|----------|
| 行星减速机 | 3-100 | 94-98% | 低 |
| 谐波减速机 | 30-320 | 70-90% | 极低 |
| 蜗轮蜗杆 | 5-100 | 40-90% | 中 |
| 摆线针轮 | 6-119 | 90-95% | 低 |

### 3.4 惯量计算

#### 3.4.1 转盘负载

```typescript
// 转盘惯量 (实心圆盘)
J_table = m × (D × 10⁻³)² / 2  // kg·m²

// 转换为电机轴
J_load = J_table / i²  // kg·m²
```

#### 3.4.2 滚筒负载

```typescript
// 滚筒惯量 (空心圆柱)
J_drum = m × (D × 10⁻³)² / 4  // kg·m² (薄壁近似)

// 转换为电机轴
J_load = J_drum / i²  // kg·m²
```

#### 3.4.3 偏心负载（重力臂）

```typescript
// 偏心质量惯量
J_eccentric = m × (Lg × 10⁻³)²  // kg·m²

// 转换为电机轴
J_load_eccentric = J_eccentric / i²  // kg·m²
```

### 3.5 扭矩计算

#### 3.5.1 加速扭矩

```typescript
// 角加速度
α_load = a_angular  // rad/s² (负载侧)
α_motor = α_load × i  // rad/s² (电机侧)

// 加速扭矩
T_accel = (J_motor + J_load) × α_motor / η  // N·m
```

#### 3.5.2 摩擦扭矩

```typescript
// 轴承/导轨摩擦
T_friction = Tf / (i × η)  // N·m
```

#### 3.5.3 重力扭矩（偏心负载）

```typescript
// 重力产生的扭矩 (随角度变化)
T_gravity_load = m × g × (Lg × 10⁻³) × sin(θ)  // N·m (负载侧)

// 转换为电机轴
T_gravity = T_gravity_load / (i × η)  // N·m

// 最恶劣工况 (θ = 90°)
T_gravity_max = m × g × (Lg × 10⁻³) / (i × η)  // N·m
```

## 4. 直接驱动 (DIRECT_DRIVE)

### 4.1 物理模型

```
    直驱电机 → 负载转盘/直线平台 (无减速机构)
```

### 4.2 输入参数

| 参数 | 符号 | 单位 | 必填 | 默认值 | 范围 |
|------|------|------|------|--------|------|
| 驱动类型 | - | - | 是 | 旋转 | 旋转/直线 |
| 负载质量 | m | kg | 是 | - | 0.1 - 100000 |
| 转盘直径 | D | mm | 旋转 | - | 10 - 10000 |
| 行程 | S | mm | 直线 | - | 10 - 10000 |
| 机械效率 | η | % | 否 | 95% | 80% - 99% |

### 4.3 惯量计算

#### 4.3.1 旋转直驱 (DDR)

```typescript
// 转盘惯量
J_load = m × (D × 10⁻³)² / 8  // kg·m²

// 总惯量 (无减速，直接相加)
J_total = J_motor + J_load  // kg·m²
```

#### 4.3.2 直线直驱 (DDL)

```typescript
// 直线电机等效惯量
J_equiv = m × (v_max / ω_max)²  // kg·m²
// 或使用力常数直接计算
```

### 4.4 扭矩/力计算

#### 4.4.1 旋转直驱

```typescript
// 加速扭矩
T_accel = J_total × α  // N·m

// 角加速度
α = (2π × n) / (60 × t_accel)  // rad/s²
```

#### 4.4.2 直线直驱

```typescript
// 加速力
F_accel = m × a  // N

// 峰值推力需求
F_peak = F_accel + F_friction + F_external  // N
```

## 5. 同步带/皮带传动 (BELT)

### 5.1 物理模型

```
    电机 → 主动轮 ──皮带── → 从动轮 → 负载
              ↑___________↓
```

### 5.2 输入参数

| 参数 | 符号 | 单位 | 必填 | 默认值 | 范围 |
|------|------|------|------|--------|------|
| 负载质量 | m | kg | 是 | - | 0.1 - 5000 |
| 主动轮直径 | d1 | mm | 是 | - | 10 - 500 |
| 从动轮直径 | d2 | mm | 否 | =d1 | 10 - 500 |
| 皮带长度 | L | mm | 否 | - | 100 - 10000 |
| 皮带质量 | mb | kg/m | 否 | 0.1 | 0.01 - 5 |
| 机械效率 | η | % | 否 | 85% | 70% - 95% |
| 张紧力 | Ft | N | 否 | 0 | 0 - 5000 |

### 5.3 惯量计算

```typescript
// 负载惯量 (转换为电机轴)
J_load = m × (d1 × 10⁻³ / 2)²  // kg·m²

// 皮带惯量
J_belt = mb × L × 10⁻³ × (d1 × 10⁻³ / 2)²  // kg·m²

// 总惯量
J_total = J_load + J_belt  // kg·m²
```

### 5.4 扭矩计算

```typescript
// 加速扭矩
T_accel = J_total × α / η  // N·m

// 皮带张力扭矩 (恒定)
T_tension = Ft × (d1 × 10⁻³ / 2) / η  // N·m

// 摩擦扭矩 (轴承等)
T_friction = 0.05 × T_accel  // 估算值
```

## 6. 齿条齿轮传动 (RACK_PINION)

### 6.1 物理模型

```
    电机 → 减速机 → 小齿轮 ──→ 齿条 → 龙门/滑台
```

### 6.2 输入参数

| 参数 | 符号 | 单位 | 必填 | 默认值 | 范围 |
|------|------|------|------|--------|------|
| 负载质量 | m | kg | 是 | - | 10 - 50000 |
| 小齿轮分度圆直径 | d | mm | 是 | - | 20 - 500 |
| 减速比 | i | - | 否 | 1 | 1 - 50 |
| 机械效率 | η | % | 否 | 90% | 70% - 95% |
| 摩擦系数 | μ | - | 否 | 0.05 | 0.01 - 0.3 |
| 安装角度 | θ | ° | 否 | 0 | -90 to 90 |

### 6.3 惯量计算

```typescript
// 等效转动惯量
J_load = m × (d × 10⁻³ / 2)² / i²  // kg·m²

// 齿轮惯量 (估算)
J_gear = 0.1 × J_load  // kg·m²

// 总惯量
J_total = J_load + J_gear  // kg·m²
```

### 6.4 扭矩计算

```typescript
// 切向力
F_tangent = m × a + μ × m × g × cos(θ) + m × g × sin(θ)  // N

// 扭矩
T_load = F_tangent × (d × 10⁻³ / 2)  // N·m (负载侧)
T_motor = T_load / (i × η)  // N·m (电机侧)
```

## 7. 摆臂/连杆机构 (LINKAGE)

### 7.1 说明

摆臂和连杆机构运动复杂，扭矩随位置变化。建议采用**等效惯量法**或**最大力矩点法**简化计算。

### 7.2 简化模型

对于简单摆臂：

```typescript
// 臂长
L_arm = L  // m

// 负载惯量 (质心在末端)
J_load = m × L²  // kg·m²

// 最大重力扭矩 (水平位置)
T_gravity_max = m × g × L  // N·m

// 建议增加 50% 安全系数
T_design = 1.5 × T_gravity_max
```

### 7.3 建议

对于复杂连杆机构，建议：
1. 使用仿真软件（如 Adams、Simulink）获取扭矩曲线
2. 输入最大扭矩点和RMS扭矩到选型工具
3. 或采用保守估算：T_peak = 2-3 × 理论计算值

## 8. 通用计算公式

### 8.1 RMS等效扭矩

```typescript
// 典型运动周期
// 加速段(t1) → 恒速段(t2) → 减速段(t3) → 停顿段(t4)

Trms = √[(T1²×t1 + T2²×t2 + T3²×t3) / (t1+t2+t3+t4)]  // N·m

其中：
- T1 = 加速段扭矩 (含摩擦、重力)
- T2 = 恒速段扭矩 (主要是摩擦、重力)
- T3 = 减速段扭矩 (可能为负，取绝对值)
- T4 = 停顿时间
```

### 8.2 峰值扭矩

```typescript
Tpeak = max(|T1|, |T2|, |T3|)  // N·m

注意：减速时若扭矩为负，表示进入再生制动状态
```

### 8.3 转速计算

```typescript
// 最大转速
n_max = (v_max × 60) / (Pb × 10⁻³) × i  // rpm (丝杠)
n_max = ω_load × 60 / (2π) × i  // rpm (旋转)

// RMS转速 (用于热校验)
n_rms = √[(n1²×t1 + n2²×t2 + n3²×t3) / (t1+t2+t3)]  // rpm
```

### 8.4 再生能量估算

```typescript
// 减速段释放的动能
E_decel = 0.5 × J_total × (ω1² - ω2²)  // J

// 若考虑重力做功 (垂直轴下降)
E_gravity = m × g × h  // J

// 总再生能量
E_regen = E_decel + E_gravity  // J

// 制动功率
P_brake = E_regen / t3  // W
```

## 9. 参数验证规则

### 9.1 通用验证

| 参数 | 验证规则 | 错误提示 |
|------|----------|----------|
| 质量 | m > 0 | "负载质量必须大于0" |
| 速度 | v > 0 | "运行速度必须大于0" |
| 加速度 | a > 0 | "加速度必须大于0" |
| 效率 | 0 < η ≤ 1 | "效率必须在0-100%之间" |

### 9.2 丝杠专用验证

| 参数 | 验证规则 | 警告提示 |
|------|----------|----------|
| 导程 | Pb > 0 | - |
| 长径比 | L/d < 50 | "长径比过大，可能存在共振风险" |
| 临界转速 | n < ncritical | "超过临界转速，请检查丝杠选型" |

### 9.3 减速机专用验证

| 参数 | 验证规则 | 错误提示 |
|------|----------|----------|
| 减速比 | i ≥ 1 | "减速比必须大于等于1" |
| 回程间隙 | 根据精度要求 | "减速机精度可能不足" |
